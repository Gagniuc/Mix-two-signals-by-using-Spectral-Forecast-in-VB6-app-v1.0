# Spectral Forecast equation VB6 app 1.0

Spectral Forecast equation VB6 app 1.0

Here we can see different cases with two different waveform signals that are combined depending on the value of a so-called distance d. This distance d is defined from zero to the maximum value found above the two vectors that represent these signals.
